# Search

Markdown Manager includes semantic search powered by AI embeddings, letting you find documents by meaning rather than just exact keywords.

## Using Semantic Search

1. Click the **search icon** (magnifying glass) in the toolbar
2. A search input expands — type your query
3. Results appear automatically as you type (with a short debounce delay)
4. Each result shows the document name, folder path, and a relevance score bar (percentage)
5. Click a result to open that document immediately

## How It Works

Semantic search uses AI-generated embeddings to understand the meaning of your query and match it against your document content. This means:

- Searching for "deployment instructions" will find documents about deploying, even if they don't contain those exact words
- Searching for "how to set up the database" will find relevant setup guides
- Results are ranked by semantic relevance, not just keyword frequency

## Tips

- Use natural language queries — "documents about project planning" works better than single keywords
- The search covers all your documents across all categories
- Press **Escape** or click outside the search to close it
- Results show a relevance percentage to help you gauge how closely each document matches

## AI Chat Search

For more interactive search, use the **AI Chat** in "All Docs" mode. The chat assistant uses the same semantic search engine but can have a conversation with you — asking follow-up questions, comparing documents, and providing summaries of what it finds.
