# Keyboard Shortcuts

Quick reference for all keyboard shortcuts available in Markdown Manager.

## Editor Shortcuts

| Shortcut | Action |
|---|---|
| **{{mod}}+S** | Save document |
| **{{mod}}+B** | Bold selected text |
| **{{mod}}+I** | Italicize selected text |
| **{{mod}}+/** | Toggle comment |
| **{{mod}}+.** | Quick Fix — show spell check and lint suggestions at cursor |
| **{{mod}}+Shift+L** | Toggle markdown linting on/off |
| **{{mod}}+Z** | Undo |
| **{{mod}}+Shift+Z** | Redo |
| **{{mod}}+F** | Find in document |
| **{{mod}}+H** | Find and replace |
| **{{mod}}+{{alt}}+N** | New document |

## Chat Shortcuts

| Shortcut | Action |
|---|---|
| **Enter** | Send message |
| **Shift+Enter** | New line in chat input (without sending) |

## General

| Shortcut | Action |
|---|---|
| **Escape** | Close search, close modals, dismiss overlays |

## Right-Click Context Menu (Editor)

Right-clicking in the editor provides additional options:

- **Fix: Remove Trailing Spaces** — cleans up trailing whitespace
- **Fix: Remove Multiple Blank Lines** — collapses consecutive blank lines
- **Fix: Add Space After Heading Hash** — fixes heading formatting
- **Show Rule Documentation** — opens documentation for a lint rule
- **Toggle Linting** — enable/disable linting from the context menu
