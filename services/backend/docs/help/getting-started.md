# Getting Started

Welcome to Markdown Manager — a full-featured markdown editor with AI assistance, real-time collaboration, GitHub integration, and more.

## First Steps

1. **Create an account** — click "Sign Up" in the user menu (top-right) and fill in your details.
2. **Log in** — enter your credentials. If you've enabled two-factor authentication, you'll be prompted for your TOTP code.
3. **Create your first document** — open the **File** menu (top-left) and choose **New Document**. A new untitled document opens in the Drafts category.
4. **Start writing** — the editor supports full Markdown syntax with live preview. Use the split view to see your rendered output side-by-side.
5. **Save** — press **Ctrl+S** or use **File → Save**. The first time you save a Draft, you'll be asked to give it a name and assign it to a category.

## Interface Overview

The interface has four main areas:

- **Toolbar** (top) — file operations, formatting, search, sharing, AI chat, and user menu
- **Editor** (left) — Monaco-powered markdown editor with syntax highlighting, autocomplete, and spell check
- **Preview** (right) — live-rendered view of your markdown, including Mermaid diagrams, code highlighting, and math equations
- **Chat Drawer** (right side panel) — AI assistant for asking questions about your documents

## Layout Modes

- **Split View** — editor and preview side-by-side with a draggable divider
- **Editor Only** — full-width editor without the preview pane
- **Fullscreen Preview** — expand the preview to fill the screen (toolbar button)
- **Pop-out Preview** — open the rendered preview in a separate browser window
- **Mobile** — on small screens, toggle between "Edit" and "Preview" tabs

## Light and Dark Theme

Click the theme toggle in the user menu or toolbar to switch between light and dark mode. Your preference is saved automatically.
