# AI Chat Assistant

Markdown Manager includes a built-in AI chat assistant that can answer questions about your documents, summarize content, improve your writing, and more.

## Opening the Chat

Click the **chat bubble icon** in the toolbar (right side) to open the chat drawer. On mobile, use the overflow menu → **AI Chat**.

## Chat Scopes

The chat has two scope modes, controlled by the toggle at the top of the drawer:

### All Docs
Searches your entire document library using semantic embeddings to find relevant content. Use this when you want to:
- Ask questions that span multiple documents
- Find information across your library ("Which document talks about deployment?")
- Get a summary of what's in your library

You can optionally filter by **category** using the category dropdown that appears in All Docs mode.

### Current Doc
Uses only the currently open document as context. Use this when you want to:
- Ask questions about the document you're reading
- Summarize the current document
- Get help improving or restructuring it

## Context Controls

| Toggle | Description |
|---|---|
| **Deep Think** | Available in Current Doc mode only. Sends the full document text to the AI for more thorough analysis instead of using chunked embeddings. Best for complex questions about the entire document. |
| **Document Only** | Restricts the AI to only use information from your documents — it won't supplement with general knowledge. Useful when you need answers strictly from your content. |

### Editor Selection Context

If you select text in the editor before asking a question, the selected text appears as a context pill in the chat input. The AI will focus its answer on that selection. Click the × on the pill to dismiss it.

## Quick Actions

When a document is open, four quick-action buttons appear above the chat input:

- **Summarize** — generates a concise summary of the document
- **Expand Shorthand** — rewrites abbreviations and shorthand into full prose
- **Improve Structure** — restructures the document with better headings and organization
- **Fix Grammar** — corrects grammar while preserving the original meaning

## Choosing an AI Provider and Model

Click the provider bar below the scope toggle to open the model picker:

- **Left column** — lists your configured providers: Ollama (local), OpenAI, xAI (Grok), GitHub Models, Google Gemini
- **Right column** — lists available models for the selected provider, with details like context window size and parameter count
- Your selection persists across sessions

To add or manage API keys for cloud providers, go to **Settings → AI Providers**.

Ollama runs locally and requires no API key — it's the default provider.

## Working with Responses

### Inserting AI Output into Your Document

Each assistant response has action buttons:

- **Insert at cursor** — inserts the response at your current editor cursor position
- **Replace selection** — replaces the currently selected text in the editor
- **Replace document** — replaces the entire document content (requires confirmation)
- **Append to document** — adds the response to the end of the document
- **Copy to clipboard** — copies the response text

If the response contains multiple sections (separated by horizontal rules), a dropdown lets you choose which section to use for the above actions.

### Editing and Regenerating

- **Resend** — re-sends the last question to generate a fresh response
- **Edit** — edit a previous question in-line, then automatically re-send
- **Version navigation** — if you've regenerated a response, use the previous/next arrows to browse between versions

### Document Links

When the AI mentions a document by name, the name is clickable. A mini-menu appears letting you **Open** the document or **Chat About** it (switches to that document in Current Doc mode).

## Chat History

Click the **clock icon** at the top of the chat drawer to open the Chat History panel:

- Browse previous conversations with titles, provider icons, message counts, and dates
- Click a conversation to restore it with all messages and response versions
- Delete individual conversations
- Conversation titles are auto-generated by the AI after the first exchange
- Click **New Chat** to start a fresh conversation

Your active conversation persists across page reloads.

## Response Metrics

Below each assistant message, you can see timing information:
- First token latency, total generation time, token count, and which model was used
